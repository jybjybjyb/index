既来之，则安之。

**夫天地者，万物之逆旅；光阴者，百代之过客。**
**而浮生若梦，为欢几何？古人秉烛夜游，良有以也。**
况阳春召我以烟景，大块假我以文章；会桃李之芳园，序天伦之乐事。
群季俊秀，皆为惠连；吾人咏歌，独惭康乐。幽赏未已，高谈转清。
开琼筵以坐花，飞羽觞而醉月。
不有佳作，何伸雅怀？
如诗不成，罚依金谷酒数
